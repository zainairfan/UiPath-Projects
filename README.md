# 🤖 UiPath Automation Projects

Welcome to my **UiPath Automation Repository**!  
This repository showcases a collection of real-world automation workflows built using UiPath to streamline repetitive tasks and improve efficiency.

---

## 🚀 About

This repository reflects my practical experience with **Robotic Process Automation (RPA)** using UiPath.  
Each project demonstrates automation of different types of tasks including web automation, data processing, OCR, and system interactions.

---

## 📂 Projects Included

### 🔍 Google Search Automation
Automates search queries on Google and extracts relevant results efficiently.

### 📝 Notepad Automation
Performs automated typing, editing, and file operations in Notepad.

### 📄 PDF OCR & Regex Task
Extracts text from PDF files using OCR and processes data using Regular Expressions.

### 🛒 Walmart Scraping Bot
Scrapes product data from Walmart website including names, prices, and details.

### 📊 Excel Report Consolidation
Merges and processes multiple Excel files into a single structured report.

### 🌐 Website Automation Project
Automates interactions with websites such as form filling, navigation, and data extraction.
